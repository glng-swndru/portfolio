Saya merupakan lulusan Teknologi Informasi yang saat ini sedang fokus membangun karier di bidang Cloud Computing dan DevOps Engineering. Meskipun pengalaman kerja saya sebelumnya berada di luar bidang IT, saya aktif mengembangkan kompetensi teknis melalui pembelajaran mandiri dan proyek-proyek pribadi.

Proyek Saya

Simple Messaging App

Teknologi: Go, MySQL, MongoDB, AWS, GitHub Action, ELK Stack

Mengembangkan aplikasi messaging real-time ringan dengan fokus pada backend dan infrastruktur. Dibangun menggunakan Go (Golang) dengan framework Fiber untuk menangani otentikasi pengguna (via MySQL) dan pertukaran pesan real-time (via WebSockets, disimpan di MongoDB). Aplikasi dideploy di AWS (EC2, RDS) dengan pipeline CI/CD menggunakan GitHub Actions serta implementasi monitoring dan logging terpusat via ELK Stack.

Music Catalog API

Teknologi: Go, Go Fiber, PostgreSQL, Spotify API, Docker

Aplikasi web berbasis Go yang memungkinkan pengguna mengelola preferensi musik dan aktivitas mereka, dengan integrasi mendalam ke Spotify API. Fitur utama meliputi manajemen pengguna (registrasi, login), pencatatan aktivitas suka/tidak suka pada lagu, pencarian lagu via Spotify API, dan rekomendasi lagu personal. Data pengguna dan aktivitas disimpan secara persisten menggunakan PostgreSQL, dengan backend yang diekspos melalui RESTful API. PostgreSQL dijalankan menggunakan Docker untuk kemudahan setup.
