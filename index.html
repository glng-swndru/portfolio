Tentang Saya

Saya adalah lulusan Teknologi Informasi yang sedang berfokus membangun karier di bidang Cloud Computing dan DevOps Engineering. Meskipun pengalaman kerja sebelumnya berada di luar bidang IT, saya aktif memperdalam keterampilan teknis melalui proyek pribadi dan pembelajaran mandiri.

Saat ini saya mendalami layanan AWS, praktik Infrastructure as Code dengan Terraform, serta membangun aplikasi dengan Golang. Saya percaya bahwa teknologi yang efektif lahir dari kombinasi antara penguasaan teknis dan pemahaman proses bisnis.

Portofolio ini saya buat sebagai bentuk dokumentasi belajar dan kontribusi saya dalam ekosistem teknologi yang terus berkembang.
